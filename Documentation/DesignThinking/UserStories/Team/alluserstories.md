

# TFLCoMentor Platform — User Stories Table

## Admin User Stories

| UseCase ID  | Use Case                                                                                                                         |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------- |
| US_ADMIN_01 | As an Admin, I want to view the overall platform statistics so that I can quickly understand platform activity and growth.       |
| US_ADMIN_02 | As an Admin, I want to see statistics of assessments so that I can understand how many assessments are ongoing or unassigned.    |
| US_ADMIN_03 | As an Admin, I want to see all assessments in the system so that I can manage and assign them to students.                       |
| US_ADMIN_04 | As an Admin, I want to assign an assessment to multiple students so that their skills can be evaluated.                          |
| US_ADMIN_05 | As an Admin, I want to delete an assessment so that outdated or incorrect assessments can be removed.                            |
| US_ADMIN_06 | As an Admin, I want to view the summary of an assessment so that I can analyze participation and performance.                    |
| US_ADMIN_07 | As an Admin, I want to see each student's assessment status so that I can track who attempted or missed the assessment.          |
| US_ADMIN_08 | As an Admin, I want to reschedule an assessment so that students who missed it can attempt it again.                             |
| US_ADMIN_09 | As an Admin, I want to view platform session statistics so that I can monitor platform usage.                                    |
| US_ADMIN_10 | As an Admin, I want to view all session logs so that I can audit platform activity.                                              |
| US_ADMIN_11 | As an Admin, I want to view session logs of a specific student so that I can analyze their login activity.                       |
| US_ADMIN_12 | As an Admin, I want to see all platform users so that I can manage students, mentors, and employers.                             |
| US_ADMIN_13 | As an Admin, I want to view a user profile so that I can understand their role and details.                                      |
| US_ADMIN_14 | As an Admin, I want to assign or modify user roles so that users can participate in multiple roles like Student, Mentor, or SME. |
| US_ADMIN_15 | As an Admin, I want to remove a user so that inactive or invalid accounts are removed from the system.                           |
